## 功能说明：

- 🔗 直播间抢福袋功能，支持扫描平台所有直播间，并过滤出正在发放福袋的直播间进行参与
  
- 📁 支持查看中奖纪录，包括中奖时间，直播间地址，中奖物品的名称和中奖数量
- 🔄 中奖提醒，中奖时会听到“等灯等灯”音效
- 🎉 可绑定钉钉通知，中奖后将中奖信息发送至钉钉群
- 🐍 支持抢福袋，超级福袋，实物福袋，支持抢福袋同时开启抢红包
- 📦 可配置直播间过滤条件，支持直播间扫描频率及预留直播间数配置，
- 🛠 可配置使用CK轮流扫描（直播间质量会比较精准）
- 🖥 可配置停止条件，抢多少个福袋便停止或者抢多久就停止（防止风控）
- ⚙️ 可配置是否开启外部浏览器（使用指纹信息）
- 🎨 支持抢福袋后直播间停留，支持随机互动（随机点赞，评论，关注 增加账号活跃度）
- 🚨 允许多号同时进入直播间（大包直播间）
- 🙌 支持直播间中奖算法，优先抢高概率，高价值福袋（预计）
- 📜 如果福袋中奖率过高，账号可能会被官方风控，一段时间内不会再中福袋（一般1-2天）
